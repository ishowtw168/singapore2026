## 照片筆記

### #01 — 柱腳環繞式熱帶植栽

**檔名：** `landscape-ref-01-column-planter.jpg`  
**拍攝地點：** Mandai Wildlife Reserve 入口大廳（仿洞穴岩壁造景區）  
**拍攝日期：** 2026-07-05

**設計特色：**
- 圓柱基座環繞式植栽設計，以不規則灰白色石塊圍出自然邊界
- 植栽高低層次分明：高層（散尾葵）→ 中層（白鶴芋）→ 低層（花葉萬年青、紅掌）
- 融入仿岩洞的室內空間，營造「建築內的自然」氛圍
- 石材圍邊採不規則堆疊，模仿自然岩石河床感
- 地面為紅色大理石紋理，植栽的綠色形成強烈對比

**植物辨識：**
| 植物 | 位置 | 特徵 |
|------|------|------|
| 散尾葵（Areca Palm / Dypsis lutescens） | 高層中央 | 羽狀複葉，叢生，高約 1.5-2m |
| 白鶴芋（Peace Lily / Spathiphyllum） | 中層 | 大型深綠葉片，橢圓形 |
| 花葉萬年青（Dieffenbachia） | 低層前緣 | 白綠相間斑葉 |
| 紅掌（Anthurium） | 低層點綴 | 紅色佛焰苞花，增添色彩 |
| 竹芋類（Calathea） | 低層 | 葉片有明顯脈紋 |

**設計啟發：**
- 適合室內大廳、商業空間柱腳美化
- 石塊圍邊取代花盆，更自然融入空間
- 熱帶植物組合適合台灣氣候（高溫高濕）
- 層次設計可套用於庭院角落或入口造景

### #02 — 岩壁前獨立棕櫚叢植

**檔名：** `landscape-ref-02.jpg`  
**拍攝地點：** Mandai Wildlife Reserve 入口大廳（仿岩洞空間）  
**拍攝日期：** 2026-07-05

**設計特色：**
- 大型獨立棕櫚作為主景，搭配蕨類地被形成自然叢林感
- 背景為仿天然岩壁（灰色石灰岩質感），頂部有攀附植物垂掛
- 無花盆、無圍邊，植物直接種植於地面花床，邊界自然過渡
- 整體呈現「洞穴口的熱帶雨林」意象
- 層次：高層棕櫚（3m+）→ 中層蘇鐵/棕櫚幼株 → 低層蕨類地被

**植物辨識：**
| 植物 | 位置 | 特徵 |
|------|------|------|
| 圓扇棕櫚（Fan Palm / Livistona） | 高層主角 | 圓形掌狀葉片，叢生茂密，約 3m 高 |
| 蘇鐵（Cycad） | 中層右側 | 羽狀複葉，質感厚實 |
| 腎蕨（Nephrolepis / Boston Fern） | 低層前緣 | 大量蕨類覆蓋地表 |
| 鐵線蕨類 | 低層 | 細緻羽狀葉 |
| 岩壁攀附植物 | 背景頂部 | 自然垂掛於岩壁上方 |

**設計啟發：**
- 適合庭院角落或圍牆前的獨立景觀焦點
- 棕櫚＋蕨類組合 = 低維護熱帶風情
- 灰色石材牆面（或水泥牆）可模仿此效果
- 地被蕨類取代草皮，適合遮蔭處
- 不需圍邊，讓植栽自然擴展更顯野趣

### #03 — 建築轉角棕櫚＋地被毯植設計

**檔名：** `landscape-ref-03.jpg`  
**拍攝地點：** Mandai Wildlife Reserve 建築走廊轉角（岩壁＋樓梯旁）  
**拍攝日期：** 2026-07-05

**設計特色：**
- 建築物轉角處的三角形植栽花床，善用畸零空間
- 亮綠色地被植物大面積鋪滿，形成「綠色地毯」效果
- 棕櫚＋落葉小喬木（細枝型）搭配，營造疏密對比
- 背景有仿岩壁洞穴與粉色建築立面，植栽柔化了硬質建材
- 花床邊緣切齊地面，無明顯圍邊，乾淨俐落

**植物辨識：**
| 植物 | 位置 | 特徵 |
|------|------|------|
| 棕竹（Rhapis / Lady Palm） | 中層主角 | 掌狀分裂葉，叢生，約 1.5m |
| 細枝落葉小喬木 | 高層右側 | 細長枝條，疏透樹冠，提供輕盈的垂直線條 |
| 卷柏 / 翠雲草（Selaginella） | 地被層 | 亮黃綠色，密集覆蓋，形成毛茸茸質感 |
| 腎蕨（Nephrolepis） | 地被層間 | 零星穿插於卷柏之間 |

**設計啟發：**
- 亮綠色地被（卷柏/翠雲草）可替代草皮用於半遮蔭環境
- 適合建築轉角、樓梯旁、走廊邊等畸零空間
- 簡單的「地被毯 + 1-2 棵中型植物」即可營造豐富感
- 棕竹耐蔭，適合台灣室內外半日照環境
- 不需過多種類，色彩對比（深綠棕竹 vs 亮綠地被）就很有效果

### #04 — 瀑布意象蕨類混植花床（Fount of Life）

**檔名：** `landscape-ref-04.jpg`  
**拍攝地點：** Mandai Wildlife Reserve「Fount of Life」入口區（靈感來自爪哇 Madakaripura 瀑布）  
**拍攝日期：** 2026-07-05

**設計特色：**
- 主題為「瀑布旁的植物生態」，模擬潮濕環境植栽
- 以各種蕨類為主角，搭配耐濕闊葉植物，營造雨林地被層
- 設有解說牌（Fount of Life / Watermark），具教育功能
- 無明顯花盆或容器，植栽直接種於邊坡花床
- 多層次混植：樹蕨（高）→ 中型蕨類 → 闊葉地被 → 禾本科/吊蘭類
- 全綠色系，不同深淺綠色＋不同葉形製造層次

**植物辨識：**
| 植物 | 位置 | 特徵 |
|------|------|------|
| 桫欏/樹蕨（Tree Fern / Cyathea） | 高層左側 | 大型羽狀葉，木質化莖幹 |
| 腎蕨（Nephrolepis） | 中層 | 長條羽狀複葉 |
| 黃金葛 / 綠蘿（Epipremnum aureum） | 中低層 | 心形綠葉，蔓性 |
| 白鶴芋（Spathiphyllum） | 中層 | 深綠大葉 |
| 禾本科/蜘蛛蘭類 | 低層前緣 | 線形細長葉片 |
| 觀音蓮 / 姑婆芋類（Alocasia） | 中層 | 大型箭形葉 |

**設計啟發：**
- 「瀑布旁濕地」主題造景：全蕨類＋耐濕植物，適合台灣多雨環境
- 樹蕨作為焦點大植物，不需開花就有強烈存在感
- 純綠色系設計靠「葉形差異」創造視覺豐富度
- 教育解說牌的設計可參考用於私人庭園導覽
- 適合水池邊、溪流旁、北面牆下等潮濕角落

### #05 — 設備遮蔽式植栽（機械美化）

**檔名：** `landscape-ref-05.jpg`  
**拍攝地點：** Mandai Wildlife Reserve — Rainforest Wild Adventure WEST 入口旁  
**拍攝日期：** 2026-07-05

**設計特色：**
- 用植栽環繞遮蔽大型戶外設備（空調室外機/通風設備）
- 不規則深灰色碎石塊圍邊，與 #01 同手法但石色不同（灰黑色調）
- 植栽巧妙遮蓋設備底部，讓醜陋的機械融入環境
- 背靠仿岩壁，整體看起來像「岩石旁長出的植物」
- 實用性設計：美化 + 不妨礙設備通風散熱

**植物辨識：**
| 植物 | 位置 | 特徵 |
|------|------|------|
| 散尾葵（Areca Palm） | 高層中央 | 亮黃綠色羽狀葉，約 1.5m |
| 竹芋類（Calathea / Ctenanthe） | 中低層主體 | 大面積鋪排，白綠條紋葉脈明顯 |
| 花葉萬年青（Dieffenbachia） | 中層 | 白綠斑葉 |
| 紅掌（Anthurium） | 低層點綴 | 紅色苞片增色 |
| 變葉木 / 彩葉植物 | 低層 | 增加色彩變化 |

**設計啟發：**
- **超實用案例！** 用植栽美化冷氣室外機、水塔、電箱等設備
- 石塊圍邊半高（約 15-20cm），不阻擋設備氣流
- 竹芋類大面積使用效果好，葉片大、遮蔽力強
- 選擇耐熱植物（設備會散熱）：散尾葵、竹芋都耐熱
- 適合台灣住家外牆旁冷氣主機美化
- 成本低、維護簡單、效果立竿見影

### #06 — 連續花槽彩葉屏障植栽

**檔名：** `landscape-ref-06.jpg`  
**拍攝地點：** Bird Paradise 園區走廊（有頂棚的遮蔭步道旁）  
**拍攝日期：** 2026-07-05

**設計特色：**
- 深銅色/鏽鐵色方形大花槽連續排列，形成走廊側邊的綠色屏障
- 花槽材質為耐候鋼（Corten Steel），顏色與植物的紅銅色調呼應
- 以「彩葉植物」為主角，非傳統全綠色：紅色、銅色、黃綠色交錯
- 高中低層次在花槽內完成，無需地面種植
- 整排連續感創造韻律，走道空間感被植栽柔化
- 兼具圍欄功能，遮蔽後方設施

**植物辨識：**
| 植物 | 位置 | 特徵 |
|------|------|------|
| 紅竹/朱蕉（Dracaena marginata / Cordyline） | 高層主角 | 紅銅色細長線形葉，放射狀，非常搶眼 |
| 斑葉榕（Ficus / 斑葉品種） | 高層 | 黃綠色斑葉，圓形樹冠 |
| 吊蘭 / 銀紋草（Chlorophytum / Liriope） | 低層 | 細長線形綠葉，下垂覆蓋花槽邊緣 |
| 麥門冬（Ophiopogon） | 低層 | 深綠色密生草狀 |
| 蕨類 | 低層點綴 | 亮綠色羽狀葉 |

**設計啟發：**
- **耐候鋼花槽**質感高級，適合現代風格庭院或陽台
- 彩葉植物（紅色系）取代開花植物，四季有色彩不需等花期
- 連續排列花槽 = 移動式綠籬，可隨時調整位置
- 適合頂樓露台、走廊、停車場邊界等需要屏障的場所
- 朱蕉/紅竹耐旱耐曬，維護超低
- 花槽高度約 60-80cm，站立時恰好遮蔽視線

### #07 — 熱帶水景＋竹構造園藝

**檔名：** `landscape-ref-07.jpg`  
**拍攝地點：** Bird Paradise 園區內（鳥舍水景展示區）  
**拍攝日期：** 2026-07-05

**設計特色：**
- 自然風水池＋石橋＋竹製水車裝置，完整的東南亞鄉村水景造景
- 竹圍欄＋竹水桶＋竹製敲水裝置（添水/鹿威し風格），增添動態聲音元素
- 水池邊緣以不規則石塊堆砌，小碎石鋪面過渡到草地
- 背景為椰子樹林，營造熱帶島嶼氛圍
- 水岸邊種植禾本科水生草，自然且柔化水池硬邊
- 石橋連接小島，創造空間層次和動線趣味

**植物辨識：**
| 植物 | 位置 | 特徵 |
|------|------|------|
| 椰子樹（Coconut Palm） | 背景高層 | 多棵成林，熱帶代表 |
| 禾本科水生草 / 狼尾草 | 水岸邊 | 線形葉片叢生，柔化水池邊緣 |
| 闊葉灌木 | 前景低層 | 綠色橢圓葉 |
| 地被草皮 | 小島表面 | 覆蓋石橋旁的土丘 |

**構造物：**
- 竹圍欄（麻繩綁結）
- 竹製水桶＋支架
- 竹製敲水/添水裝置（類似日式「鹿威し」但東南亞風格）
- 不規則石塊砌石橋

**設計啟發：**
- 竹＋石＋水 的組合 = 低成本高效果的自然風水景
- 竹構造在台灣取材容易，可 DIY
- 添水裝置增加「動態聲音」，庭院更有生命力
- 水池＋石橋＋小島的組合適合中型庭院（5坪以上）
- 禾本科水生草取代規則修剪的水岸，維護更省力
- 整體風格：東南亞鄉村 / 峇里島度假風
