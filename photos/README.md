# 📸 旅遊照片 Travel Photos

此資料夾供所有成員上傳新加坡旅遊照片，大家都可以讀取和瀏覽。

## 上傳方式

1. 直接將照片檔案放入此資料夾（或依日期建子資料夾）
2. 建議依日期命名子資料夾，例如：
   - `07-04/` — 抵達日
   - `07-05/` — 動物園
   - `07-06/` — 聖淘沙 Luge
   - `07-07/` — 環球影城
   - `07-08/` — 自由行
   - `07-09/` — 離境日

## 命名建議

- 檔名盡量有描述性，例如：`zoo-panda-family.jpg`
- 支援格式：`.jpg` `.jpeg` `.png` `.heic`

## 注意事項

- ⚠️ 此 repo 為公開倉庫，上傳的照片所有人都看得到
- 請勿上傳包含護照、信用卡、訂單編號等敏感個資的照片
- GitHub 單檔限制 100MB，建議照片壓縮後再上傳
- 若照片太多，可考慮使用 Google Drive 或 iCloud 共享相簿作為替代方案

## 瀏覽方式

所有成員可直接在 GitHub 上瀏覽：
